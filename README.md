## vimconfig
	该目录下，存放了我常用的三种vim配置。

	vimrc_base是无插件的版本，用于临时场合下使用vim。
	
	vimconfig是结合网上的配置，自己修改后的一个版本。含有常用的插件。
	
	myconfig是自己配置的vim版本，目前主要使用这个版本vim配置。含有常用的插件。
